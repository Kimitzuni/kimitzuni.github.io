Workflow

My workflow is mostly in the terminal, with programs such as Vim (well actually NeoVIM), mutt, git and other general terminal programs. I search through code with the vgrep program, which is a much better alternative to grep, and I highly recommend it. I have 3 laptops, because I like to make things overly awkward I guess. These laptops are:

  • Lenovo ThinkPad T460s w/ 128GB NVMe, 8GB RAM, Intel i5-6200U (View on Lenovo's Site)
  • Lenovo ThinkPad X230 w/ 320GB SSD, 12GB RAM, Intel i5-3200U (View on Lenovo's Site)
  • HP Pavilion 15s-eq2510sa w/ 512GB NVMe, 8GB RAM, AMD Ryzen 7 5700U (View on Hewlett Packard's Site)
Let's start with the T460S. On this I run openSUSE Tumbleweed with the XFCE desktop environment. The desktop environment is just my personal preference over the other environments Tumbleweed provides (GNOME and KDE Plasma). The reason for using openSUSE is simply because that's what happened to be written on my USB at the time I decided to reinstall Linux.

The X230 has Arch Linux, with the i3 Window Manager and a kind of "minimalist" setup, ie. that it's mostly terminal operated. However I haven't updated that system in a while, so it probably is quite outdated, just the nature of running rolling release distributions.

And finally is the HP Pavilion, which is also running the same as the T460S, but with some differences, mainly this has disk encryption, and because I have the space to do this on this machine - a separate /home partition, which is roughly 480GB.

My main machine is built from random hardware that I had, and is mostly focused on storage space and quietness. It has 16GB of DDR4 RAM, and Intel Core i5-9400F, NVIDIA GeForce GTX 1650 from MSI, a Corsair CX550F RGB PSU, and the motherboard is "MS-7C09" according to neofetch, but a readable name for it is the MSI H310M PRO-VDH Plus. In terms of storage, the main boot drive is a 240GB Maxtor Z1 SSD, and for extra storage there is a 4TB Seagate BarraCuda drive that I've had for about 3 years, and only recently managed to fill 300GB up. This machine also runs openSUSE Tumbleweed with XFCE and has disk encryption.

As for web browsers, because people like to know, I can use any web browser, but my preference is Mozilla Firefox, with (obviously) the extension uBlock Origin. Because you would be absolutely insane to not use uBlock

My configuration files are currently not public, however I am working on making them public by removing all things I don't want people to have, such as my neomutt configuration having my real email address, SMTP URL etc. When they are ready, they will be uploaded to my GitHub.
