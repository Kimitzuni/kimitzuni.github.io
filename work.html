2020

(3/20 - 7/20) Online Introduction to C# Programming

During March to July of 2020, I setup, organized and ran an online introduction to C# Programming course, designed to teach people the basics of C# and code etiquette. The course spanned 4 months, each lesson (or session) happening on the Wednesday of each week, going from writing simple "Hello, World!" programs to creating more advanced applications, such as a file manager, a music player and an image viewer. Along with that, other things that this course went over was;

  • Using NuGet to install extra packages
  • Porting Pre-existing Programs to C#
  • Using Git and Git Services (ie. GitHub/GitLab)
  • Reading Hardware Data (ie. XInput Controllers)
