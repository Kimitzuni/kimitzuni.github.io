Maybe, Ubuntu Isn't That Bad

Saturday, 7th January 2023

In a previous blog post, I complained about Canonical's Ubuntu distribution. However, I think that I got some things wrong in that post, and I want to correct some of those things. If you want to read the original blog post, you can do - Ubuntu Is A Bad Distribution.

Table of Contents

  • Ubuntu's Flavours
  • Steam on Ubuntu
    • Issues
  • Recommended Ubuntu Distros

Ubuntu's Flavours

In the previous blog post, I focused primarily on Ubuntu. Stock, vanilla plain Ubuntu that you can download from Ubuntu's Website. GNOME 40. That version. However, there are other "flavours" of Ubuntu that have different desktop environments, such as Ubuntu with MATE - Ubuntu MATE, or Ubuntu with KDE Plasma - Kubuntu. And I highly recommend the Kubuntu distribution, it's one of the nicest feeling versions of Ubuntu. I don't know how to explain it, but it feels very nice. But in this post, I will be focusing on Ubuntu Unity, which is the flavour of Ubuntu that uses the old Unity desktop environment from pre-18.04, and I just prefer this desktop over GNOME. So, I have this distribution installed on all my machines, except for the Ryzen Laptop, and I have been using it for a couple of weeks, and here's what I have to say about this over Ubuntu.

It feels much... faster, a lot more responsive, even with Snap packages. The slowness is, sometimes still there however it is a lot less prevalent.

Steam on Ubuntu

Steam on Ubuntu is a lot better than Steam on openSUSE, as it works. I was completely wrong on the performance of games on Ubuntu, and specifically my take on running Half-Life 2 on Ubuntu being unplayable. This was plain wrong. My aim is to not misinform people, and I use the blog as a place to express myself, and talk about things I find interesting, but this was outright wrong, and I'm sorry for that. Here is the actual performance of Half-Life 2 on Ubuntu: Exactly the same as on Windows. You can play it. It's not laggy at all. There is occasionally some lag, but mostly between loading zones, and even then I put that down to - It's loading stuff. Other games, such as Black Mesa, the Half-Life: Source remake, perform equally as well. There is some lagging with that, but it is a more graphics intensive game.

Issues

There are some issues with Steam on Ubuntu, and this mostly has to do with the suspend/sleep function of the distribution - Once leaving suspend, the UI of the Steam program just breaks. It's still usable, but broken. I have no idea why it does this, but other programs like Chrome also break in the same way.

Recommended Ubuntu Distros

I do recommend some Ubuntu and Ubuntu based distributions, such as Kubuntu and Ubuntu Unity, however there is another called Linux Mint, which uses the Cinnamon Desktop Environment and provides a more Windows-esque experience.
