My name is Rebecca, I'm known for being "that one girl who keeps telling you about Arch Linux." Alternatively, I'm a mathematics and computer science nerd, vim and GNU Emacs user, and someone with an odd obsession with the ThinkPad brand.

I use GNU/Linux, more specifically I use Arch Linux. I have used Linux since 2012, but used it primarily since 2017. Generally my workflow consists of terminal programs, so that includes neovim or vim for text editing, ncmpcpp for music, and calcurse as a calendar application.

I have my dotfiles (config files) available on GitHub for anyone to use, and as a backup for myself.

I can code in various programming languages, my preferred being C. I'm not sure why I like coding in C so much, I just do for some reason. I have experience with Python, C++, Rust, Go, QB64 (Modern QBASIC). I also use LaTeX to write documents, because I find that it's more efficient than opening a big office suite, such as LibreOffice, and I don't have to move my hands off of the keyboard.
