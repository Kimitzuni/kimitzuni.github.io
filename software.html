Google Death Guesser

GitHub Repository | View Site

Google has a reputation of making products and services end-of-life or end of support, instead of just maintaining them. Such as YouTube annotations. The explanation for this was "the mobile app doesn't support them", but you make the app Google. So I made a (albeit) bad game based on this. You are shown a Google product, and told to guess when it will or did get killed. It is written in mostly JavaScript.

Credman

GitHub Repository

credman is a password manager that I made whilst learning Java. I would not recommend using this to store actual passwords, as I didn't put that much consideration into security of the storage of the passwords, and they are stored in just plain text. Use something like KeePassXC
